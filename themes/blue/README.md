# Blue theme helpers

## Chráněný obsah v článku
Do textu obsahu můžete vložit blok viditelný jen přihlášeným pomocí značek `[logged] ... [/logged]` (případně `[members_only] ... [/members_only]`).

```html
<p>Úvod článku, který uvidí všichni.</p>

[logged]
<p>Tato část je pouze pro přihlášené čtenáře.</p>
[/logged]
```

Nepřihlášeným se místo skrytého obsahu automaticky zobrazí informační hláška s odkazem na přihlášení.
